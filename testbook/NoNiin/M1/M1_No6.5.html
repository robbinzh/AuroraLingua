句 1
Tänään on Iinan ja Chrisin siivouspäivä.
1. 语块拆解
Tänään on / Iinan ja Chrisin / siivouspäivä.今天是 / Iina和Chris的 / 清洁日/大扫除日。
2. 变位与要点
语法要点:
• 属格 (Genetiivi): Iinan ja Chrisin。表示“谁的”。
• 复合词 (Yhdyssana): siivouspäivä = siivous (清洁/打扫，名词) + päivä (日子)。siivous 源自动词 siivota。
词汇变位与原型:
• Tänään: 副词 (今天)。
• siivouspäivä: 原型 siivouspäivä。词干 siivouspäivä-。
今天是 Iina 和 Chris 的大扫除日。
句 2
He siivoavat koko kodin putipuhtaaksi!
1. 语块拆解
He siivoavat / koko kodin / putipuhtaaksi!他们打扫 / 整个家 / (变得)一尘不染！
2. 变位与要点
语法要点:
• 宾语 (Objekti): koko kodin。这是单数全格宾语（属格形式 -n），因为“打扫整个家”是一个有明确结果、被彻底完成的动作。
• 变格词 (Translatiivi -ksi): putipuhtaaksi。
• 规则: 变格 (Translatiivi) 表示“状态的改变”或“变成某种状态”。
• 含义: 家被打扫后，变成了干净的状态。
• 词汇: putipuhdas 是一个强调词，意为“干干净净”、“一尘不染”。
词汇变位与原型:
• siivoavat: 原型 siivota (4型)。词干 siivoa-。规则: -t- 脱落，a 加倍 -> he siivoavat。
• kodin: 原型 koti (家)。词干 kodi- (弱级 t->d)。单数属格/宾格。
• putipuhtaaksi: 原型 putipuhdas。词干 putipuhtaa- (s结尾词的变化)。变格: putipuhtaa + ksi.
他们把整个家打扫得一尘不染！
句 3
Ensin Iina pesee pyykkiä.
1. 语块拆解
Ensin / Iina pesee / pyykkiä.首先 / Iina洗 / 衣服/脏衣物。
2. 变位与要点
语法要点 (重点对比):
• 部分格宾语 (Partitiivi): pyykkiä。
• 为什么用部分格？ 此时强调的是“她在洗衣服”这个动作过程，或者指洗“一些”衣服，而不是强调“把这堆衣服洗完”的结果。
词汇变位与原型:
• pyykkiä: 原型 pyykki。词干 pyykki-。单数部分格 pyykki + ä。
首先，Iina 开始洗衣服。
句 4
Pyykkikori on nimittäin täpötäynnä.
1. 语块拆解
Pyykkikori on / nimittäin / täpötäynnä.洗衣篮是 / 原来(你瞧) / 满满当当的。
2. 变位与要点
语法要点:
• 状态格 (Essiivi -na): täpötäynnä。
• 规则: 状态格表示“处于某种状态”。
• 词汇: täpötäysi (挤满的/塞满的)。
• 连接词: nimittäin (即，因为，你瞧)。用于解释原因或补充说明。
词汇变位与原型:
• täpötäynnä: 原型 täpötäysi。词干 täpötäyte- (通常i结尾形容词) 但这里是特殊的 si -> nne ? 不，täysi 的词干是 täyte-，但作为 Essiivi 有特殊保留形式，或是基于辅音词干 täyn- + nä。täpö-：这是一个加强前缀（vahvistussana），类似于英语中的 "cram-" 或 "stone-"（如 stone-cold）。它本身没有实际含义，也不能独立变位，作用就是把后面的形容词程度推到极致。- täynnä：这是形容词 täysi（满的）的状态格（Essive Case）形式。 täpötäynnä 是状态格，不可变位，单数复数一样。如果你想表达“在满满的篮子里”，你会回到原形 täysi 去变位（例如：täpötäydessä korissa），但作为表语描述“满了”这个状态时，永远只用 täpötäynnä。
原来洗衣篮已经塞得满满当当了。
句 5
Iina laittaa pyykit koneeseen.
1. 语块拆解
Iina laittaa / pyykit / koneeseen.Iina放 / 那些衣服 / 进机器里。
2. 变位与要点
语法要点 (重点对比):
• 全格宾语 (Totaaliobjekti): pyykit (复数主格)。
• 为什么不用部分格了？ 刚才还在洗 pyykkiä，现在变成了 pyykit。因为这里指的是“把那些（特定的、篮子里的）衣服”全部放进去，动作有明确对象和结果。
• 入格 (Illatiivi): koneeseen (进机器)。laittaa + Illatiivi 是固定搭配，表示“放入某处”。
词汇变位与原型:
• pyykit: 原型 pyykki。词干 pyyki- (弱级 kk->k)。复数主格。
• koneeseen: 原型 kone。词干 konee-。入格 seen (e结尾单词的入格形式)。
Iina 把衣服放进洗衣机。
句 6
Sillä aikaa, kun kone pesee pyykit, Iina pyyhkii pölyt rätillä.
1. 语块拆解
Sillä aikaa, / kun kone pesee pyykit, / Iina pyyhkii pölyt / rätillä.在那段时间， / 当机器洗衣服(的时候)， / Iina擦灰尘 / 用抹布。
2. 变位与要点
固定搭配: Sillä aikaa, kun... (在...的同时 / 当...的时候)。
语法要点:
• 接格 (Adessiivi) 表工具: rätillä。用抹布。
• 宾语: pyykit (全格，指机洗的这批衣服)；pölyt (全格，复数主格，指把灰尘擦掉，通常用复数)。
词汇变位与原型:
• pesee: 原型 pestä (3型)。词干 pese-。
• pyyhkii: 原型 pyyhkiä (1型)。词干 pyyhki-。KPT变化 hk 保持不变。
• pölyt: 原型 pöly (灰尘)。
• rätillä: 原型 rätti (抹布)。词干 räti- (弱级 tt->t)。接格。
在洗衣机洗衣服的时候，Iina 用抹布擦灰尘。
句 7
Kun kone on pessyt pyykit, Iina ripustaa ne kuivumaan pyykkitelineelle.
1. 语块拆解
Kun kone on pessyt / pyykit, / Iina ripustaa ne / kuivumaan / pyykkitelineelle.当机器已经洗完 / 衣服， / Iina挂它们 / 去变干 / 到晾衣架上。
2. 变位与要点
语法要点:
• 现在完成时 (Perfekti): on pessyt。表示动作已经完成。
• 宾语: ne (它们，指代 pyykit)。复数宾语的全格形式（对于代词 ne 是宾格）。
• 第三不定式入格 (-maan): kuivumaan。
• 规则: 动词 ripustaa (挂) 后面接另一个动作表示目的或状态变化时，用第三不定式入格。挂起来 -> 去干。
• 向格/接格 (Allatiivi): pyykkitelineelle。挂到架子上。
词汇变位与原型:
• pessyt: 原型 pestä。过去分词形式 (NUT-partisiippi)。
• ripustaa: 原型 ripustaa (1型)。
• kuivumaan: 原型 kuivua (变干)。词干 kuivu- + maan.
• pyykkitelineelle: 原型 pyykkiteline。复合词 pyykki + teline (架子)。词干 telinee- (e结尾词)。
当洗衣机洗完衣服后，Iina 把它们挂到晾衣架上晾干。
句 8
Iinaa harmittaa, että heillä ei ole kuivausrumpua.
1. 语块拆解
Iinaa harmittaa, / että heillä ei ole / kuivausrumpua.Iina感到懊恼/烦心， / 因为他们没有 / 烘干机。
2. 变位与要点
语法要点 (核心句型):
• Partitiiviverbi (部分格动词) 作主语: harmittaa (使...烦恼)。
• 这种“心理感受动词”通常要求体验者用 部分格 (Partitiivi)。即：烦恼正在“搞” Iina。
• 结构: Partitiivi (体验者) + Verbi (单三)。-> Iinaa harmittaa.
• 否定拥有句: heillä ei ole + 部分格。
• kuivausrumpua (部分格)。因为是否定句。
词汇变位与原型:
• Iinaa: 原型 Iina。部分格。
• harmittaa: 原型 harmittaa (1型)。
• kuivausrumpua: 原型 kuivausrumpu。复合词 kuivaus (干燥) + rumpu (鼓/滚筒)。词干 rumpu- (强级 mp)。此处部分格 rumpua。
Iina 有点烦恼，因为他们没有烘干机。

Chris ottaa / siivouskomerosta / silitysraudan ja -laudan /Chris拿 / 从清洁柜里 / 熨斗和(熨衣)板 /

语法要点:

• 连字符用法: silitysraudan ja -laudan。

句 12

语法要点:

• Paistinpannun: 原型 paistinpannu。复合词 paistin (paisti的属格，烤肉的) + pannu (锅)。
• käsin: 原型 käsi。固定副词用法。

2.1 总结该段在全文中的意义和作用

这是一篇极佳的生活场景叙述文。

  1. 词汇积累: 它涵盖了家庭大扫除（Siivouspäivä）的核心词汇：洗衣 (pyykki)、熨烫 (silittää)、洗碗 (tiskata)、灰尘 (pölyt)、家电 (kone, kuivausrumpu)。
  2. 语法功能: 它像教科书一样展示了宾语格的变化（根据动作是过程还是结果，是部分还是全部）以及动态地点格（从哪拿出来 -sta，放到哪去 -an）的精确用法。

语法点补充1 täpötäynnä

täpötäynnä（塞得满满当当）这个词其实是一个复合固定形式，它的“变位”逻辑非常有意思，因为它实际上是一个已经变好位的状态词。

在芬兰语中，这类词通常被称为“僵化形式”或“固定副词性形容词”。我们来拆解它的构造：

1. 词源拆解：täpö + täynnä

  • täpö-：这是一个加强前缀（vahvistussana），类似于英语中的 "cram-" 或 "stone-"（如 stone-cold）。它本身没有实际含义，也不能独立变位，作用就是把后面的形容词程度推到极致。

  • täynnä：这是形容词 täysi（满的）的状态格（Essive Case）形式。

2. 为什么它看起来不像一般的变位？

在芬兰语中，描述“状态”时经常使用状态格（后缀 -na/nä）。

  • 原形：täysi（满）

  • 状态格：täynnä（处于满的状态）

当你看到 täpötäynnä 时，它已经处在状态格中了。在句子 "Pyykkikori on täpötäynnä" 中，它的功能是描述洗衣篮当前的状态。这种结构与你在图片中看到的 "Selkä on kipeä"（背是疼的） 或 "Iho on kuuma"（皮肤是热的） 非常相似，都是“主语 + on + 描述状态的词”。

3. 它需要进一步变位吗？

不需要。 这是一个非常高效的词：

  • 它是不可变位的（Invariable）。无论主语是单数还是复数，它都保持 täpötäynnä。

    • Pyykkikori on täpötäynnä. (洗衣篮是满满的。)

    • Pyykkikorit ovat täpötäynnä. (洗衣篮们是满满的。)

  • 如果你想表达“在满满的篮子里”，你会回到原形 täysi 去变位（例如：täpötäydessä korissa），但作为表语描述“满了”这个状态时，永远只用 täpötäynnä。

4. 补充：句中的 nimittäin

顺便提一下，句中的 nimittäin 是一个非常有用的填充词，意思是“即”、“因为”、“你要知道”。

  • 它通常用来解释前一句话的原因。

  • 例子：En voi pestä vaatteita. Pyykkikori on nimittäin täpötäynnä. (我没法洗衣服。因为你看，洗衣篮已经塞得满满当当了。)
