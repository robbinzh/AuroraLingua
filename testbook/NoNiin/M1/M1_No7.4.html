句 1
Chris lähtee, mutta Hamsa juttelee vielä Kaisan ja Sandrinen kanssa.
1. 语块拆解
Chris lähtee, / mutta Hamsa juttelee / vielä / Kaisan ja Sandrinen kanssa.Chris 离开， / 但是 Hamsa 聊天 / 仍然/还 / 和 Kaisa 与 Sandrine 一起。
2. 变位与要点
• 句子结构: 并列句。前半句主语 Chris，谓语 lähtee；后半句主语 Hamsa，谓语 juttelee，状语 Kaisan ja Sandrinen kanssa。
• 语法要点:
• Postpositio (后置词) kanssa:
• 规则: 意为“和...一起”。它前面的名词或人名必须用 属格 (Genetiivi, -n)。
• 逻辑: 属于谁的伴随。
• 举例: Kaisan kanssa (Kaisa-n + kanssa)。注意这里 Kaisan ja Sandrinen 都是属格。
• 词汇变位与原型:
• lähtee: 原型 lähteä (1型动词)。词干 lähde- (注意 t-d 强弱阶变化，此处是弱级)。
• juttelee: 原型 jutella (3型动词)。词干 juttele- (注意 tt-t 强弱阶变化，3型动词此处用强级)。
Chris 离开了，但 Hamsa 还在和 Kaisa 以及 Sandrine 聊天。
句 2
Hamsa: Mitä te aiotte tehdä?
1. 语块拆解
Mitä / te aiotte / tehdä?什么 / 你们打算 / 做？
2. 变位与要点
• 语法要点:
• Aikoa + Infinitive (打算做某事):
• 规则: 动词 aikoa (打算) 后面总是紧跟动词的 第一不定式 (A-infinitivvi/原型)。
• 用法: 这是芬兰语表达将来意图的最常用方式之一。
• 词汇变位与原型:
• aiotte: 原型 aikoa (1型)。词干 aio- (注意 k-0 消失，k在两个元音间消失)。
• tehdä: 动词原型 (2型)。注意它是不规则动词，变位时词干是 tee- (如 minä teen)，但这里直接用原型。
Hamsa: 你们打算做什么？
句 3
Kaisa: Ei mitään erikoista.
1. 语块拆解
Ei / mitään erikoista.不 / 任何 特别的。
2. 变位与要点
• 语法要点:
• 否定句中的部分格 (Partitiivi):
• 逻辑: 这是一个省略句，完整的句子是 "Me emme tee mitään erikoista" (我们不做任何特别的事)。否定词 ei 后面，宾语通常变成部分格。
• mitään 是 mikään (任何) 的部分格。
• erikoista 是形容词 erikoinen (特别的) 的部分格。
• 词汇变位与原型:
• erikoista: 原型 erikoinen (-nen结尾的词)。词干 erikoise-。部分格变化规则：-nen -> -sta。
Kaisa: 没什么特别的。
句 4
Me vaan istutaan ja jutellaan.
1. 语块拆解
Me vaan / istutaan / ja jutellaan.我们只是 / 坐着 / 和聊天。
2. 变位与要点
• 语法要点 (重点 - 口语语法):
• Me + 被动式 (Passiivi):
• 规则: 在标准语 (Kirjakieli) 中应该说 Me istumme ja juttelemme。但在口语 (Puhekieli) 中，第一人称复数 (我们) 几乎总是用 被动式 来代替。
• 结构: Me + [动词被动式]。
• vaan 是 vain (只/仅仅) 的口语形式。
• 词汇变位与原型:
• istutaan: 原型 istua (1型)。被动式构成：词干 istu- + -taan。
• jutellaan: 原型 jutella (3型)。被动式构成：原型去掉-la + -laan。
我们就坐着聊聊天。
句 5
Kohta me lähdetään kauppaan viemään yks bonuskorttihakemus.
1. 语块拆解
Kohta / me lähdetään / kauppaan / viemään / yks bonuskorttihakemus.一会儿 / 我们出发 / 去商店 / 去送 / 一个 积分卡申请表。
2. 变位与要点
• 语法要点:
• Lähteä + mihin + tekemään (去某地做某事):
• 链式结构: 动词 lähteä (出发) + 目的地 (入格/Illatiivi) + 动作 (第三不定式入格/-maan)。
• 逻辑: "出发" 是一个动作，动作的方向是 "进入商店"，目的是 "进入送东西这个动作里"。
• Kauppaan: kauppa (商店) + an (入格)。
• Viemään: viedä (带/送) -> vie (词干) + mään (第三不定式入格)。
• 数词口语: yks = yksi。
• 词汇变位与原型:
• lähdetään: 原型 lähteä。口语形式 (Me + 被动式)。注意这里保留了强级 d，因为被动式标记前的音节结构。
• bonuskorttihakemus: 复合词。Bonus + kortti (卡) + hakemus (申请)。原型 hakemus，词干 hakemukse-。
待会儿我们去趟商店交一张积分卡申请表。
句 6
Sandrine: Minä menen illalla asemalle. Mä tapaan siellä yhden minun kaverin.
1. 语块拆解
Minä menen / illalla / asemalle.我去 / 在晚上 / 去车站。
Mä tapaan / siellä / yhden minun kaverin.我见 / 在那里 / 一个我的朋友。
2. 变位与要点
• 语法要点:
• Asemalle (向格/Allatiivi): 词根是 asema。通常去车站、广场等开放式场所用 -lle (去...上面/边上)，而不是 -an (入格)。虽然现在很多人也把车站视为封闭建筑，但习惯用法依然是 asemalle。
• Tavata 的宾语格: Tavata (遇见) 是及物动词，宾语通常用 属格形式的宾格 (单数事物完成时/整体时)。这里是 yhden kaverin (一个朋友，n结尾)。
• 口语代词: Mä = Minä。
• 词汇变位与原型:
• tapaan: 原型 tavata (4型)。词干 tapaa- (注意 v-p 强弱变化，4型动词在此处变强)。
• yhden: 原型 yksi。属格/宾格形式。
Sandrine: 我晚上要去火车站。我在那里见我的一个朋友。
句 7
Hän tulee Turusta.
1. 语块拆解
Hän tulee / Turusta.他来 / 从图尔库。
2. 变位与要点
• 语法要点:
• Tulla + Mistä (出格/Elatiivi): 动词“来”要求接“从哪里来”。
• Turusta: Turku + sta。注意 k-0 的强弱变化消失了 (Turku -> Turu -> Turusta)。
他从图尔库来。
句 8
Hamsa: Mikä on Turku?
1. 语块拆解
Hamsa: Mikä on Turku?
2. 变位与要点
• 疑问结构： Mikä on X? 用于询问“X 是什么”。
• Mikä 是疑问代词主格；on 是 olla 的 3单。
• 专有名词 Turku 在此作表语主格，不变格。
Hamsa: 图尔库是什么？
句 9
Kaisa: Hih! Turku. Se on kaupunki Länsi-Suomessa.
1. 语块拆解
Hih! Turku. / Se on kaupunki / Länsi-Suomessa.嘻！图尔库。 / 它是城市 / 在西芬兰。
2. 变位与要点
• 语法要点:
• 方位: Länsi-Suomessa (Inessiivi/内格, -ssa)。表示“在...里面”。这是一个复合词，前面 Länsi (西) 不变化，后面 Suomi 变成 Suomessa。
Kaisa: 嘻！图尔库。那是西芬兰的一个城市。
句 10
Hamsa: Ok. Turku. Kyllä minä sen tiedän.
1. 语块拆解
Ok. Turku. / Kyllä minä / sen tiedän.好。图尔库。/ 当然我 / 它 知道。
2. 变位与要点
• 语法要点:
• 宾格代词: 动词 tietää (知道) 的宾语。代词 se (它) 的宾格是 sen。
• Kyllä: 这里表示强调，“我是知道的”。
• 词汇变位与原型:
• tiedän: 原型 tietää (1型)。词干 tiedä- (注意 t-d 强弱变化)。
Hamsa: 好吧，图尔库。我当然知道它。
句 11
Turun murre on hauska, mutta vaikea.
1. 语块拆解
Turun murre on hauska, mutta vaikea.
2. 变位与要点
• 语法要点:
• 属格属性: Turun murre = 图尔库的方言。Turun 是 Turku 的属格 (k消失)。
图尔库方言很有趣，但是很难。
句 12
Kaisa: Joo, murteet on vaikeita ymmärtää.
1. 语块拆解
Joo, / murteet on / vaikeita / ymmärtää.对， / 方言们是 / 难的 / 去理解。
2. 变位与要点
• 语法要点:
• 主谓一致 (口语): 主语 murteet 是复数，标准语应该用 ovat，口语常用单数 on。
• 补语的部分格: 当主语是复数 (murteet)，形容词补语通常用部分格复数 -> vaikeita (原型 vaikea)。
• 结构: 形容词 + 动词不定式。 Vaikea(a/ita) + ymmärtää (难理解)。
Kaisa: 是啊，方言很难懂。
句 13
Hamsa: Onko sinun kaveri kauan täällä Kirjalassa?
1. 语块拆解
Hamsa: Onko sinun kaveri kauan täällä Kirjalassa?
2. 变位与要点
• (注: Kirjala 是地名，Kirjalassa 是内格)
• 是非问句： Onko ... ? = On + 疑问后缀 -ko。
• 地点格： Kirjalassa 为内格，表示“在 Kirjala”。
• 时间副词： kauan 表示“多久/很久”，询问停留时长。
Hamsa: 你的朋友要在基尔亚拉 (Kirjala) 待很久吗？
句 14
Sandrine: Pari päivää. Sunnuntaina hän matkustaa taas junalla takaisin Turkuun.
1. 语块拆解
Pari päivää.一两天/几天。
Sunnuntaina / hän matkustaa / taas / junalla / takaisin Turkuun.在周日 / 他旅行 / 又 / 乘火车 / 回 到图尔库。
2. 变位与要点
• 语法要点:
• Pari + Partitiivi: 词汇 pari (一对/两个/少许) 后面跟的名词必须是 部分格单数。-> päivää (原型 päivä)。
• Sunnuntaina (爱格/Essiivi): 当表示“在星期几”这一整天时，用 -na。
• Junalla (接格/Adessiivi): 交通工具用 -lla (在...上面/凭借...)。
• Takaisin + Mihin: Takaisin (回) 隐含方向，所以地点用入格 -> Turkuun (回图尔库去)。
Sandrine: 待几天。周日他又会坐火车回图尔库。
句 15
Hamsa keskustelee vielä hetken Sandrinen ja Kaisan kanssa. Sitten hän lähtee kahvilasta kuntosalille. Naiset jäävät vielä kahvilaan.
1. 语块拆解
Hamsa keskustelee / vielä hetken / Sandrinen ja Kaisan kanssa.Hamsa 交谈 / 再一会儿 / 和 Sandrine 与 Kaisa。
Sitten / hän lähtee / kahvilasta / kuntosalille.然后 / 他离开 / 从咖啡馆 / 去健身房。
Naiset jäävät / vielä / kahvilaan.女人们留下 / 仍然 / 在咖啡馆。
2. 变位与要点
• 语法要点 (关键):
• Hetken (时间宾格): Hetki (片刻)。此处用属格形式表示持续了一段时间。
• Lähteä + Mistä + Mihin: 从哪离开 (Elatiivi, kahvilasta) 去哪里 (Allatiivi, kuntosalille)。注意 kuntosali 通常用外部格 -lle，哪怕是在室内，这是一种习惯用法（类似去某个场所/机构）。
• Jäädä + Mihin (入格/Illatiivi): 这是超级重点！
• 中文逻辑我们说“留在咖啡馆里”，你会想用 kahvilassa。
• 但在芬兰语中，jäädä (停留/留下) 被视为一个带有方向性的动作（以此结束运动），所以必须回答 "Mihin?" (Into where?)。
• 正确: Jäädä kahvilaan (Staying into the cafe).
• 错误: Jäädä kahvilassa.
• 词汇变位与原型:
• jäävät: 原型 jäädä (2型)。词干 jää-。
Hamsa 又和 Sandrine 还有 Kaisa 聊了一会儿。然后他离开咖啡馆去了健身房。女士们则还留在咖啡馆里。

2.1 总结该段在全文中的意义和作用

这一段是典型的社交闲聊 (Small talk) 场景。

  1. 情节推进： 交代了每个人的后续去向（Kaisa去商店，Sandrine去车站接人，Hamsa去健身房），并引出了新地点“图尔库 (Turku)”，为可能的后续故事线（如Sandrine的朋友）做铺垫。
  2. 语言功能： 集中展示了将来计划的表达（aiotte tehdä, menen, tapaan）以及移动动词的格要求（从哪来 mistä，去哪 mihin，在哪 missä）。特别是 jäädä + 入格 的用法，是初学者必须掌握的难点。

2.2 长难芬兰语单词语法分解表

单词 (Sana) 原型 (Perusmuoto) 中文翻译 词干 (Vartalo) 词类/形式 构词法与语法深度解析 (Kiina)
Istutaan istua (我们)坐 istu- 动词 (被动式) 口语重点: 标准语是 me istumme。口语常用被动式 istutaan 代替第一人称复数。
Viemään viedä 去送/带 vie- 动词 (III不定式) 连动结构: 前面有 lähdetään (出发)，后面的动词必须变成 "第三不定式入格" (-maan/-mään)，表示移动的目的。
Bonuskorttihakemus ~hakemus 积分卡申请 ~hakemukse- 名词 (复合词) 复合词: Bonus + kortti (卡) + hakemus (申请)。<br>变格提示: -us/-ys 结尾的名词，变格时通常 s -> kse (如 hakemuksen)。
Länsi-Suomessa Länsi-Suomi 在西芬兰 ~Suome- 专有名词 (内格) 地名复合: 前缀 Länsi- (西) 不变，只有核心词 Suomi 变格。Suomi -> Suome- (i->e) + ssa。
Vaikeita vaikea 难的 vaikea- 形容词 (部分格复数) 补语一致: 主语 murteet 是复数，且无法数清/抽象，表语形容词要用部分格复数。规则: ea 结尾双元音，复数部分格变为 -eita。
Kahvilaan kahvila (留)在咖啡馆 kahvila- 名词 (入格) Reaktio (接格): 动词 jäädä (留下) 强制要求入格 (-an)。虽然中文是“在”，芬兰语逻辑是“停留在...里”。
