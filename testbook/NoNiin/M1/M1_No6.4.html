📖 Ilta kotona (家中的夜晚)

句 1
Illalla Chris tekee kotitehtävät ja Iina avaa TV:n.
1. 语块拆解
Illalla / Chris tekee / kotitehtävät /在晚上 / Chris做 / 家庭作业 /
ja Iina avaa / TV:n.并且Iina打开 / 电视。
2. 变位与要点
语法要点 (核心难点 - 宾语):
• 全格宾语 (Totaaliobjekti) 的单复数区别:
↪ 复数全格: kotitehtävät。当宾语是复数且动作也是“完成”性质时，宾语使用复数主格的形式（结尾 -t）。
↪ 逻辑: Chris 会把作业做完，不仅是一个过程，还有一个结果。
↪ 单数全格: TV:n。当宾语是单数且动作是“完成”性质时（这里指把电视打开这个动作完成），宾语使用单数属格形式（结尾 -n）。
↪ 书写规则: 缩写词（如 TV）加词尾时，需要用冒号 : 连接。
• 时间状语: Illalla (Adessiivi -lla) 表示“在晚上”。
词汇变位与原型:
• tekee: 动词 tehdä (做)。不规则动词。
↪ 词干: teke-.
• kotitehtävät: 名词 kotitehtävä (家庭作业)。
↪ 构词: koti (家) + tehtävä (任务)。
↪ 形式: 复数主格（用作宾语）。
• TV:n: 名词 TV (或 televisio)。
↪ 形式: 单数宾格（属格形式）。
• avaa: 动词 avata (打开)。
↪ 类型: 4型动词 (-ta)。
↪ 变位: 去掉 -ta，a 变长 (ava- + a)。
在晚上 / Chris做 / 家庭作业 / 并且Iina打开 / 电视。
句 2
"Häiritseekö telkkari sinuu?" Iina kysyy.
1. 语块拆解
"Häiritseekö / telkkari / sinuu?" / Iina kysyy.“打扰吗 / 电视 / 你？” / Iina问。
2. 变位与要点
语法要点:
• 动词 häiritä 的接格 (Reaktio):
↪ 规则: häiritä (打扰) 必须接 部分格 (Partitiivi)。因为打扰是一种持续的影响，或者是一种情感上的触动。
• 口语代词 (Puhekieli):
↪ sinuu = 标准书面语 sinua (你的部分格)。
↪ 现象: 在口语中，结尾的 -ua/-yä 经常合并成双元音 -uu/-yy。
↪ 举例: minua -> minuu, sinua -> sinuu.
• 口语词汇: telkkari 是 televisio 的常用口语词。
词汇变位与原型:
• Häiritseekö: 动词 häiritä (打扰)。
↪ 类型: 5型动词 (-ita/-itä)。
↪ 变位: 词干加 -se-。häirit -> häiritse-.
↪ 形式: 单三 + 疑问后缀 -kö.
• sinuu: 代词 sinä。部分格 sinua 的口语变体。
“打扰吗 / 电视 / 你？” / Iina问。
句 3
"Ei se häiritse minua. Minä olen valmis", Chris sanoo ja tulee Iinan luo.
1. 语块拆解
"Ei se häiritse / minua. /“它不打扰 / 我。
Minä olen / valmis", / Chris sanoo /我是 / 完成了的/准备好的”，/ Chris说 /
ja tulee / Iinan luo.并来到 / Iina身边。
2. 变位与要点
语法要点:
• 否定句中的宾语: Ei häiritse minua。否定句宾语强制使用 部分格 (minua)。
• olla valmis: 意思是“准备好了”或者“完成了某事”。这里指 Chris 做完作业了。
• 后置词结构 (Postpositio) - 动态:
↪ 规则: tulla (来) 是一个移动动词，所以要用表示方向的后置词结构。
↪ jonkun (属格) + luo (向...身边)。
↪ 对比: 如果是“在Iina身边”，则说 Iinan luona。
词汇变位与原型:
• häiritse: 动词 häiritä 的否定式主干。
• luo: 后置词。表示“去某人处”。
“它不打扰 / 我。 我是 / 完成了的/准备好的”，/ Chris说 / 并来到 / Iina身边。
句 4
He istuvat sohvalla ja katsovat yhdessä televisiota.
1. 语块拆解
He istuvat / sohvalla / ja katsovat / yhdessä / televisiota.他们坐 / 在沙发上 / 并看 / 一起 / 电视。
2. 变位与要点
语法要点:
• istua + 地点: 坐在某物上面，用 接格 (-lla)。sohva -> sohvalla。
• katsoa + Partitiivi: “看电视”是典型的持续性动作，宾语 televisio 必须用部分格 televisiota。
词汇变位与原型:
• istuvat: 动词 istua (坐)。1型。
• sohvalla: 名词 sohva (沙发)。
• televisiota: 名词 televisio。部分格。
他们坐 / 在沙发上 / 并看 / 一起 / 电视。
句 5
"Mitä ohjelmaa katsotaan?" Iina kysyy.
1. 语块拆解
"Mitä ohjelmaa / katsotaan?" / Iina kysyy.“什么节目 / 我们看？” / Iina问。
2. 变位与要点
语法要点 (高阶用法):
• 被动语态代替第一人称复数 (Passiivi me-muotona):
↪ 规则: 在口语和非正式语境中，katsotaan (被动语态现在时) 经常用来代替 me katsomme (我们看)。
↪ 含义: “我们要看什么节目？” 或 “咱们看什么节目？”。建议或共同行动的意味很强。
• 疑问词的格: 因为 katsoa 要求部分格，所以疑问词 mikä 和名词 ohjelma 都要变成部分格 -> Mitä ohjelmaa.
词汇变位与原型:
• Mitä: 代词 mikä。部分格。
• katsotaan: 动词 katsoa。被动语态现在时。
↪ 构词: 弱词干 katso- + 被动标记 ta + 人称/时态 an.
“什么节目 / 我们看？” / Iina问。
句 6
"Ihan sama, valitse sinä", Chris vastaa.
1. 语块拆解
"Ihan sama, / valitse sinä", / Chris vastaa.“完全一样(无所谓)，/ 你选吧”，/ Chris回答。
2. 变位与要点
固定搭配: Ihan sama。字面意是“完全相同”，实际常用作“无所谓、随便、都行”。
祈使句 (Imperatiivi): valitse 是第二人称单数祈使形式。
• 规则: 取第一人称单数 valitsen，去掉 n -> valitse.
词汇变位与原型:
• valitse: 动词 valita (选择)。
↪ 类型: 3型动词 (-ta/-tä, 前非元音)。
↪ 变位: valita -> valitse-.
“完全一样(无所谓)，/ 你选吧”，/ Chris回答。
句 7
Wilpertti nukkuu sohvalla Iinan vieressä.
1. 语块拆解
Wilpertti nukkuu / sohvalla / Iinan vieressä.Wilpertti睡 / 在沙发上 / Iina的旁边。
2. 变位与要点
语法要点:
• 后置词结构 - 静态:
↪ 规则: vieressä (在...旁边/紧挨着) 是后置词，要求前面的词用属格。
↪ Iinan (属格) + vieressä (内格形式，表示静止的位置)。
↪ 对比: 之前 Iinan luo 是动作方向，这里 Iinan vieressä 是静态位置。
词汇变位与原型:
• vieressä: 原型 vieri (旁边/侧边)。内格形式用作后置词。
Wilpertti睡 / 在沙发上 / Iina的旁边。

句 9
Yhdeltätoista myös Iina ja Chris menevät nukkumaan.
1. 语块拆解
Yhdeltätoista / myös Iina ja Chris / menevät / nukkumaan.十一点的时候 / 也 Iina和Chris / 去 / 睡觉。
2. 变位与要点
语法要点 (高阶时间表达):
• 整点时间的格:
↪ 规则: 当表示“在几点钟”发生某事时，数字使用 离格 (Ablatiivi, -lta/-ltä)。
↪ 结构: yksitoista (11)。变格时，前半部分 yksi 变为 yhdeltä，后半部分 toista 保持不变（因为它已经是部分格形式）。
↪ 结果: Yhdeltätoista.
• 第三不定式入格 (3. infinitiivi illatiivi):
↪ 规则: mennä (去) + 做某事。后面的动词必须变成“去做的状态”，即第三不定式的入格 (-maan/-mään)。
↪ 结构: nukkua -> 强词干 nukku- + maan.
↪ 含义: 去睡觉 (go to sleep)。
词汇变位与原型:
• Yhdeltätoista: 原型 yksitoista (11)。
↪ 变格分析: yhde- (yksi的弱词干) + ltä + toista.
• nukkumaan: 动词 nukkua。第三不定式入格。
十一点的时候 / 也 Iina和Chris / 去 / 睡觉。

2.1 总结该段在全文中的意义和作用

这一段通过晚间的休闲时光，展示了家庭生活的常态。

  1. 叙事推进: 从做作业（责任）过渡到看电视（休闲），最后到睡觉（结束一天）。
  2. 语言特色:
    • 口语融入: 角色对话中自然地使用了口语形式 (sinuu, telkkari, katsotaan 表示“我们看”)，而旁白保持书面语。
    • 位置与移动: 清晰地区分了“来到身边” (luo) 和“在旁边” (vieressä)。
    • 时间与动作: 引入了精确时间点 (yhdeltätoista) 和动作目的 (mennä nukkumaan)。
