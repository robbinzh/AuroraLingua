句 1
Sillä aikaa kun Iina touhuaa keittiössä, Chris alkaa siivota lattioita.
1. 语块拆解
Sillä aikaa kun / Iina touhuaa / keittiössä, /在那段时间当 / Iina忙活 / 在厨房里，
Chris alkaa / siivota / lattioita.Chris开始 / 打扫 / 地板。
2. 变位与要点
语法要点:
• 句子结构: 时间状语从句 Sillä aikaa kun... + 主句。
• Sillä aikaa kun: 固定连词词组，表示“在...的同时”。
• alkaa + 1. Infinitiivi: alkaa 后接动词原形，如 siivota。
• 对比: ruveta 常接第三不定式（如 ruveta siivoamaan）。
• 部分格宾语: lattioita 强调“开始打扫”的过程性，且是复数不定量。
词汇变位与原型:
• touhuaa: 原型 touhuta（4型），3单。
• siivota: 原型 siivota（4型），这里是第一不定式。
• lattioita: 原型 lattia，复数部分格。
当 Iina 在厨房忙活的时候，Chris 开始打扫地板。
句 2
Hän lakaisee lattialta ensin isommat roskat harjalla ja rikkakihvelillä.
1. 语块拆解
Hän lakaisee / lattialta / ensin / isommat roskat /他扫 / 从地板上 / 首先 / 较大的垃圾 /
harjalla / ja rikkakihvelillä.用刷子 / 和用簸箕。
2. 变位与要点
语法要点:
• 离格 (Ablatiivi, -lta): lattialta 表示“从表面离开”。
• 全格宾语: isommat roskat，强调扫走这些特定垃圾的结果。
• 比较级: isommat 来自 iso -> isompi。
• 工具格 (Adessiivi, -lla): harjalla、rikkakihvelillä 表示“用什么工具”。
词汇变位与原型:
• lakaisee: 原型 lakaista（3型），3单。
• isommat: 原型 iso，比较级复数主格。
• rikkakihvelillä: 原型 rikkakihveli，接格。
他先用刷子和簸箕把地板上较大的垃圾扫走。
句 3
Sitten hän imuroi ja lopuksi hän vielä moppaa lattiat.
1. 语块拆解
Sitten / hän imuroi /然后 / 他吸尘 /
ja lopuksi / hän vielä moppaa / lattiat.并且最后 / 他还拖 / 地板。
2. 变位与要点
语法要点 (核心对比):
• 全格宾语: lattiat（复数主格形式作全格宾语）强调动作完成。
• 对比句1: siivota lattioita（过程）vs moppaa lattiat（结果）。
词汇变位与原型:
• imuroi: 原型 imuroida（2型），3单。
• moppaa: 原型 mopata（4型），3单。
• lattiat: 原型 lattia，复数主格（全格宾语）。
然后他吸尘，最后他还拖了地。

• menevät: 原型 mennä（3型），3复。
• siivoamaan: 原型 siivota，第三不定式入格。

句 6

2. 变位与要点
语法要点:
• 全格宾语: ikkunat（复数）表示把窗户都洗完。
• auttaa + 部分格: häntä 是固定要求，不能用宾格。
词汇变位与原型:
• Lopuksi: 原型 loppu，变格（Translatiivi）副词化用法。
• ikkunat: 原型 ikkuna，复数主格（全格宾语）。
• häntä: 原型 hän，部分格不规则形式。
最后，Iina 把窗户也洗了，Chris 帮了她。

句 8
Mutta kyllä nyt on ihanaa, kun koti on putipuhdas!
1. 语块拆解
Mutta / kyllä nyt on / ihanaa, /但是 / 确实现在是 / 美妙的，

2. 变位与要点
语法要点:
• 抽象主语表达: on ihanaa 中表语用部分格 ihanaa。

词汇变位与原型:
• ihanaa: 原型 ihana，部分格。
• putipuhdas: 原型 putipuhdas，复合强化形容词。
但是现在家里一尘不染的，感觉真棒！

2.1 总结该段在全文中的意义和作用

这是整个大扫除故事的结局。

  1. 情节上：这一段集中描写最繁重的家务（扫地、拖地、刷厕所、擦窗）以及完成后的满足感。
  2. 语法上：这是理解“过程 vs. 结果（Aspekti）”与宾语格（Partitiivi / Totaaliobjekti）的关键段落。

语法点补充 1：过程 vs 结果（Aspekti）

  • siivota lattioita：强调“开始打扫/正在打扫”的过程，所以用部分格 lattioita。
  • moppaa lattiat：强调“拖完了地板”的结果，所以用全格宾语 lattiat。

语法点补充 2：mennä + 3. Infinitiivi Illatiivi

  • 结构：mennä + 动词词干 + maan/mään。
  • 例子：menevät siivoamaan vessan（他们去打扫厕所）。

语法点补充 3：auttaa 的格要求

  • auttaa 是典型部分格动词，后接人称代词时用部分格：hän -> häntä。
